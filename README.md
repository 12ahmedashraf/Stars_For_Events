# Stars for events

[StarsForEvent](public/stars_for_events.png)

## What's this this ?

This is an events' booking platform which I learned many things while making (every project I make I love Next.js more xD) , It contains a neat home page , simple about & an events page where you view current events (AND GOOD-LOOKING MOVING STARSS), it handles registration for events also it contains a user where user can manage his profile & view tickets also it has an admin dashboard that manages all these data!.

### Tech stack:

NEXT.JS ,Tailwind CSS, clerk for auth & supabase (BAAS)
